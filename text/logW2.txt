🖼️ Image Classification
🔹 Definition

Image Classification is a technique in Artificial Intelligence and Computer Vision where a computer analyzes an image and assigns it to a specific category or label.

👉 In simple words:
It answers the question: “What is in this image?”

🔹 Example
Image of a cat → Output: “Cat” 🐱
Image of a car → Output: “Car” 🚗
Medical scan → Output: “Tumor” or “Normal”
🔹 How Image Classification Works
1. Input Image
The system takes an image (JPG, PNG, etc.)
2. Preprocessing
Resize image
Normalize pixel values
Remove noise
3. Feature Extraction
Important features are detected:
Edges
Shapes
Colors
Textures
4. Model Processing
Uses deep learning models like:
Convolutional Neural Networks (CNN)
5. Classification
Model assigns a label based on learned patterns
🔹 Types of Image Classification
1. Binary Classification
Two classes only
👉 Example: Spam / Not Spam
2. Multi-class Classification
More than two categories
👉 Example: Cat, Dog, Car
3. Multi-label Classification
One image can have multiple labels
👉 Example: “Dog + Person + Car”
🔹 Applications
📱 Face recognition
🚗 Self-driving cars
🏥 Medical diagnosis
🛒 Product recognition
🔍 Security and surveillance
🔹 Advantages
Fast and automatic image analysis
High accuracy with deep learning
Useful in many industries
🔹 Limitations
Needs large training data
Can fail with poor-quality images
Requires high computation power
🔹 Example Workflow
Upload image
Model processes image
Output → Label (e.g., “Dog”)
🔹 Conclusion

Image classification is a powerful technology that allows computers to understand visual data. It plays a major role in modern AI systems like autonomous vehicles, healthcare diagnosis, and smart applications.

🔥 Simple Difference (Important for Your Project)
Concept	What it does
OCR	Reads text from images
Regex	Extracts patterns from text
Image Classification	Identifies object/category in image

Exposure Scoring Engine
🔹 Definition

An Exposure Scoring Engine is a system that analyzes data and calculates a risk score based on how much sensitive or private information is exposed.

👉 In simple words:
It tells “how risky or exposed the data is” by giving a score.

🔹 Purpose

The main goal of an Exposure Scoring Engine is to:

Identify sensitive information
Measure risk level
Help in decision-making (safe / unsafe)
🔹 How It Works (Step-by-Step)
1. Data Input
Takes input from:
OCR extracted text
Documents
User data

👉 Example:

Name: John
Phone: 9876543210
Aadhaar: 1234 5678 9012
2. Data Analysis
Uses techniques like:
Regex (to find patterns like phone numbers, emails)
Keyword matching (Name, DOB, ID)
3. Sensitive Data Detection
Identifies:
Personal data (Name, Address)
Confidential data (ID numbers, bank details)
4. Risk Evaluation
Assigns weight to each type of data:
Data Type	Risk Level
Name	Low
Phone Number	Medium
Aadhaar / ID	High
5. Score Calculation
Combines all risks into a final score

👉 Example:

Score = 85/100 → High Risk
6. Output Result
Displays:
Risk score
Risk level (Low / Medium / High)
Highlighted sensitive data
🔹 Example
Input:
Name: Rahul
Phone: 9876543210
Email: rahul@gmail.com
Output:
Detected:
Name → Low risk
Phone → Medium risk
Email → Medium risk

👉 Final Score: 60/100 (Medium Risk)

def calculate_exposure_score(data):
    score = 0

    if "phone" in data:
        score += 20
    if "email" in data:
        score += 20
    if "aadhaar" in data:
        score += 50
    if "name" in data:
        score += 10

    if score >= 70:
        level = "High Risk"
    elif score >= 40:
        level = "Medium Risk"
    else:
        level = "Low Risk"

    return score, level

import re

def extract_sensitive_data(text):
    phone = re.findall(r'\d{10}', text)
    email = re.findall(r'[a-zA-Z0-9._%+-]+@[a-zA-Z0-9.-]+\.[a-z]{2,}', text)

    return {
        "phone": phone,
        "email": email
    }
