# PrivacyShield — Advanced Privacy Exposure Tool
A professional desktop application for scanning, monitoring, and redacting Sensitive Personal Information (PII) to ensure complete data compliance.

### 🔍 Deep File Scanner
Instantly scan PDFs, Word documents, text files, and images for exposed PII. Findings are categorized by severity using advanced NLP (spaCy) and Computer Vision OCR.

### 🖥️ Live Screenshot Monitor
Real-time monitoring of designated application windows or entire screens. If sensitive data appears on screen, PrivacyShield instantly alerts you and logs the exposure risk.

### 🕵️‍♂️ URL & Social Media Scanner
Deep-scan live websites and public social media profiles to identify leaked email addresses, phone numbers, and other risky data points out in the wild.

### 🎭 Smart Image Blur
Automatically detects human faces and physical ID cards in images and intelligently applies a permanent redaction blur.

### ⚡ Attack Simulation Pipeline
Run hypothetical attack scenarios against your detected vulnerabilities. Understand exactly how an attacker might exploit leaked data and get actionable remediation steps.

### 🛡️ Background Agent Watcher
Set-and-forget background directory monitoring. Whenever a new file is downloaded or saved, PrivacyShield silently scans it and triggers a native Windows notification if it contains risky PII.

### 🔄 Auto-Updater
PrivacyShield ships with a built-in updater (updater.exe) that automatically checks GitHub Releases for newer versions, downloads the setup installer, and applies it silently.

### 💻 System Requirements
**OS:** Windows 10 / 11 (64-bit)
**Permissions:** Admin rights required for system-wide background agent alerts.
**Install size:** ~400 MB
**RAM:** 512 MB minimum
